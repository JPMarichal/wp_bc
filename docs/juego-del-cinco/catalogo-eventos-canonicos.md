# Catálogo de Eventos Canónicos

> Catálogo maestro de eventos canónicos que aparecen en las 5 obras canónicas.
> Cada evento tiene un ID único (`_evento_canonico`) que agrupa todas las perícopas relacionadas.
> Referencias a cualquier correlación adicional (AT, BdM, DyC, PGP) están documentadas.

## Índice

| Categoría | Cantidad |
|:----------|:--------:|
| Narrativa/Milagro | ~90 |
| Discurso/Enseñanza | ~40 |
| Parábola | ~25 |
| Profecía | ~10 |
